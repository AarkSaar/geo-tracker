📍 Installation Guide

GeoTrack Setup

Step-by-step instructions to get the app running on your computer — no prior experience needed.

1
Install Node.js
Windows Mac

Node.js is the engine that runs the development server. You only need to install this once — it stays on your computer for future projects too.

ℹ️ Go to https://nodejs.org and click the big green button that says "LTS" (Long Term Support). Download and run the installer — click Next on every screen and accept all the defaults.

To verify it worked: After installing, open a Terminal (Mac) or Command Prompt (Windows) and type:

💻 Terminal / Command Prompt
node --version

You should see something like v20.x.x. If you do, Node.js is ready.

💡 How to open a terminal on Windows: Press Windows Key + R, type cmd, press Enter. On Mac: press Cmd + Space, type Terminal, press Enter.
2
Install VS Code (Code Editor)
Windows Mac

VS Code is a free code editor made by Microsoft. You'll use it to view the project files — think of it like Microsoft Word, but for code.

ℹ️ Go to https://code.visualstudio.com, click Download for Windows (or Mac). Run the installer and accept all defaults.
💡 During installation on Windows, check the box that says "Add to PATH" if you see it — this makes it easier to use later.
3
Unzip the Project Files

You should have a file called geotrack-app.zip. Follow these steps:

📁 Windows: Right-click geotrack-app.zip → "Extract All…" → choose your Desktop or Documents folder → click Extract.

Mac: Double-click the zip file — it will automatically create a folder next to it.

You should end up with a folder containing these 4 files:

📂 geotrack-app folder
index.html
main.js
style.css
package.json
4
Open the Folder in VS Code

Open VS Code, then go to File → Open Folder… and select the geotrack-app folder you just unzipped. You'll see the 4 files listed in the left sidebar.

Now open the built-in terminal inside VS Code: go to Terminal → New Terminal from the top menu. A panel will appear at the bottom of VS Code — this is where you'll type commands.

💡 The terminal in VS Code automatically starts inside your project folder, so you don't need to navigate anywhere — just type the commands directly.
5
Install Project Dependencies

In the VS Code terminal, type this command and press Enter:

💻 VS Code Terminal
npm install

This downloads Vite (the development tool) into a node_modules folder. It only takes a few seconds and you only need to do this once per project.

⚠️ You may see some yellow warning lines — that's normal and fine. The only problem is a red line that says ERROR. If that happens, make sure you opened the correct folder (the one with package.json inside it).
6
Start the App

In the same terminal, type this and press Enter:

💻 VS Code Terminal
npm run dev

You'll see output that looks like this:

📋 Expected output
  VITE v8.x.x  ready in 200ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
💡 Open your web browser (Chrome, Edge, or Firefox) and go to http://localhost:5173 — the GeoTrack app will load. Keep the VS Code terminal running while you use the app.

To stop the server when you're done, click inside the terminal and press Ctrl + C.

7
Log In & Test the App

Use these demo credentials to log in:

On the Employee dashboard, use the "Simulate Position" buttons to move inside or outside the geofence zone — the Clock In button only activates when you're inside the boundary.

📦 All data (sessions, records) is stored in your browser's localStorage — it persists between page refreshes but stays private to your machine. To clear all data, open browser DevTools (F12) → Application → Local Storage → right-click and Clear.
🎉

You're all set!

The app is running locally on your machine. Every change you make to the code files will instantly refresh in the browser.
